ClipKitty

Your clipboard history from day one. Unlimited storage, instant fuzzy search, full multi-line previews — all private, all on your Mac.

Unlimited History

No caps, no expiry. Everything you copy is stored — text, code, links, images, colors. Find that snippet from last year in milliseconds.

Instant Fuzzy Search

Powered by Tantivy, the same engine behind production search platforms. Type "improt" and find "import". Stays fast across millions of items.

Multi-Line Preview

See full content — code blocks, emails, multi-paragraph text — in a dedicated preview pane before you paste. No more guessing from truncated snippets.

Privacy First

Runs entirely on your Mac. No cloud sync, no analytics, no telemetry. Your clipboard history never leaves your device.

Keyboard Driven

Option+Space to open. Arrow keys to browse. Return to paste. Cmd+1–9 to jump. No mouse required.

Native & Fast

Built with Swift and Rust, optimized for Apple Silicon. Runs silently in your menu bar with minimal memory and zero CPU when idle.

Install via Homebrew

Install from the command line:

brew install jul-sh/clipkitty/clipkitty
