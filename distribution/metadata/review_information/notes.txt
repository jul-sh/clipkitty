ClipKitty is a clipboard history manager that saves everything you copy and lets you quickly find and re-use it. It is available on Mac and iPhone.

HOW TO TEST (macOS):

1. Launch ClipKitty. A small cat icon appears in the menu bar (top-right of your screen).
2. Copy some text in any app (e.g. select text in Safari, press Cmd+C). Do this 2–3 times with different content.
3. Click the cat icon in the menu bar, or press ⌥Space (Option+Space).
4. Your copied items appear in a scrollable list, most recent first.
5. Use the arrow keys (Up/Down) to navigate items. The right-side preview pane shows full content.
6. Press Return to copy the selected item to the clipboard. This dismisses the panel.

HOW TO TEST (iPhone):

1. Launch ClipKitty. The main screen shows your clipboard history.
2. Copy some text in another app, then return to ClipKitty. The copied item appears in the feed.
3. Tap the + button at the bottom-right to add items manually (paste from clipboard, compose text, or import a photo).
4. Tap any item to preview it. Swipe left on an item for quick actions (delete, bookmark, edit).
5. Use the search bar (magnifying glass) to find items. Use the filter pill to filter by type (text, images, links, colors).

NOTES:
- The app has no account system and no login. All data is stored locally by default.
- Optional iCloud Sync (enabled in Settings) syncs clipboard history across your Mac and iPhone.
- On first launch, clipboard history will be empty — you need to copy content to populate it.
