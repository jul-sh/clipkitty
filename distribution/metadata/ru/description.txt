Большинство менеджеров буфера обмена удаляют историю через несколько дней или ограничивают количество элементов. ClipKitty создан для опытных пользователей, которым нужен бесконечный архив с поиском по всему, что они когда-либо копировали.

ПОЧЕМУ CLIPKITTY?

Безлимитная История — Никаких ограничений. Скопировали что-то год назад — оно всё ещё здесь. Текст, код, ссылки, цвета — ничего не истекает, ничего не обрезается.

Многострочный Просмотр — Хватит работать вслепую с однострочными сниппетами. Смотрите полные блоки кода, письма и многоабзацные тексты в специальной панели просмотра перед вставкой.

Мгновенный Поиск — Находите любой фрагмент за миллисекунды даже в библиотеке из миллионов записей. Нечёткое совпадение означает, что опечатки вас не замедлят. Напишите «improt» — найдите «import».

Приватность Прежде Всего — Ваши данные никогда не покидают Mac. Никакой облачной синхронизации, никакой аналитики, никакой телеметрии. Всё остаётся на устройстве, всегда.

Нативный и Быстрый — Разработан на Swift и Rust, оптимизирован для Apple Silicon. Работает незаметно в строке меню с минимальным потреблением памяти и нулевой нагрузкой на CPU в режиме ожидания.

Открытый Исходный Код — ClipKitty полностью открыт. Каждая строка кода публична и доступна для проверки.

УПРАВЛЕНИЕ С КЛАВИАТУРЫ

Option+Пробел, чтобы вызвать историю. Стрелки для навигации. Return для вставки. Command+1–9 для перехода. Мышь не нужна.

Примечание: В версии из Mac App Store после выбора элемента буфера обмена нажмите Command+V, чтобы вставить его в нужное приложение. Это требование обусловлено ограничениями безопасности песочницы macOS.

ClipKitty — менеджер буфера обмена, который работает так же усердно, как и вы.
