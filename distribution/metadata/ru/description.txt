Большинство менеджеров буфера обмена удаляют историю через несколько дней или хранят лишь короткий список. ClipKitty сохраняет всё и позволяет мгновенно находить нужное.

Безлимитная История — Никаких ограничений, никаких лимитов. Скопировали что-то год назад — оно всё ещё здесь. Текст, изображения, ссылки, цвета; ничего не истекает, ничего не обрезается.
Многострочный Просмотр — Точно видите, что будете вставлять, в специальной панели предпросмотра.
Мгновенный Поиск — Находите любой фрагмент за миллисекунды, будь то десять записей или десять миллионов. Создан для тех, кто много копирует и хочет всё быстро находить.
СИНХРОНИЗАЦИЯ iCLOUD
Опционально синхронизируйте историю буфера обмена на всех ваших устройствах. Включите iCloud Sync в настройках, и ваши клипы будут следовать за вами с устройства на устройство.
РАБОТАЕТ КАК ОЖИДАЕТСЯ — На Mac: ⌥Space для открытия, стрелки для навигации, Return для вставки, ⌘1-9 для перехода к любому элементу. На iPhone: нажмите для копирования, смахните для управления. Фильтруйте по типу или ищите по всему.
Безопасность и Конфиденциальность; ваши данные по умолчанию никогда не покидают компьютер. ClipKitty работает полностью локально, без телеметрии и без подписок. Опциональная синхронизация iCloud использует исключительно ваш приватный контейнер iCloud.
