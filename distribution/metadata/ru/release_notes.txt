Представляем ClipKitty.

- Безлимитная история буфера обмена — ничего не истекает, ничего не обрезается
- Мгновенный нечёткий поиск по всему архиву
- Полная многострочная панель просмотра для кода, текста и ссылок
- Управление с клавиатуры: Option+Пробел для открытия, Return для вставки
- Полная приватность — ваши данные никогда не покидают ваш Mac
