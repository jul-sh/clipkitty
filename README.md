# ClipKitty

<img src="AppIcon.icon/Assets/Image%204.png" alt="ClipKitty icon" width="30">

A fast, native clipboard manager for macOS.

<img src="[https://raw.githubusercontent.com/julsh/clipkitty/gh-pages/screenshot.png](https://github.com/jul-sh/clipkitty/blob/gh-pages/screenshot.png?raw=true)" alt="ClipKitty screenshot" width="820">

## Features

- **Instant access** – Click the menu bar icon or press ⌥Space
- **Fast search** – FTS5 trigram-powered substring matching
- **Keyboard-driven** – Arrow keys to navigate, Return to paste
- **Lightweight** – Native Swift app in the menu bar, no dock icon
