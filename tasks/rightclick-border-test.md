# Right-Click Border Screenshot Test

## Run the test

```sh
xcodebuild test -scheme ClipKittyUITests -destination 'platform=macOS' -derivedDataPath DerivedData \
  -only-testing:ClipKittyUITests/ClipKittyUITests/testRightClickBorderScreenshot
```

## Check the result

```sh
open /tmp/clipkitty_rightclick_border.png
```

## Troubleshooting

- **LFS not pulled:** If you get no items in the list, run `git lfs pull` â€” `SyntheticData.sqlite` is stored in Git LFS.
- **Code signing fails (`errSecInternalComponent`):** Run `./distribution/setup-dev-signing.sh --cleanup && ./distribution/setup-dev-signing.sh` to recreate the dev keychain.
- **Stale Xcode project:** The `.xcodeproj` is generated by Tuist. Run `tuist generate --no-open` to regenerate.
